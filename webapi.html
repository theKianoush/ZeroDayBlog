Zero-Day

MERN Stack Messaging App

By: Kian Ranjbar

Project Overview

This project is a full-stack messaging application built with the MERN stack (MongoDB, Express, React, Node.js). The app is designed to allow users to communicate seamlessly without the need for a traditional sign-up process. Users can log in directly with their Google account and start chatting immediately.

Problem Statement

I wanted to create a messaging app with robust features similar to Discord, where users can make posts and communicate within them. A key challenge was the “empty room” problem: if no one else is online, users have no one to interact with. To address this, I planned to integrate an AI chatbot for users to chat with when no friends are available.

Development Process

I started by following instructions from the book MERN Projects for Beginners by Nabendu Biwas to build a base messaging app. Implementing all the desired features took longer than expected, but the core functionality was successfully completed. I also researched ways to integrate AI for future chatbot functionality.

Key Features

  • Login via Google account – no traditional signup required.
  • Real-time messaging between users.
  • Future plans include an AI chatbot, picture messaging, custom chat rooms, message reactions, and phone notifications.

Technical Stack

  • Frontend: React
  • Backend: Node.js & Express
  • Database: MongoDB
  • Hosting: Firebase (frontend) & Heroku (backend)

Conclusion & Next Steps

This project strengthened my full-stack development skills and gave me hands-on experience building a real-time web application. Future improvements include adding an AI chatbot, media messaging, custom chat rooms, reactions, and mobile notifications to enhance user engagement.
